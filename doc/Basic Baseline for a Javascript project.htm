Setup node project

The first thing to do is to install NodeJs after this in order to setup our project probably the best this to do is to use yeoman but we will not use yeoman in order to explain the thigs that we will do

After install node. We will create one folder for our project and in this folder we will excute 

$ npm init

This command will ask us a bunch of questions and when finish it will create a package.json file. In this file npm will store information about our project and his dependencies.

After setup our project we will install bower and grunt telling that they are developer dependencies (not necessary for deploy buf if for developing

$ npm install bower grunt --save-dev

This command will update our package.json with these two dependencies and will download it in a node_modules folder
